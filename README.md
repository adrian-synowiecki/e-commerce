## General Information

E-commerce shop built with React By making this project, I aimed to learn Next.js, which is used for 
building a modern server-side applications, that are friendly
to search engine crawlers, which result in better search engine optimization.



## Technologies Used
- React
- Next.js
- Typescript
- Styled Components
- Jest
- React Testing Library
- Framer Motion
- Material UI
- Redux Toolkit
- PostgreSQL
- PrismaORM
- DatoCMS
- GraphQL
- NextAuth


## Features
List the ready features here:
- Responsivity
- Products Suggestions
- Products Grouping
- Products Filters
- Products Sorting
- Shopping Cart
- Users Orders list with action status 
- Products Searching
- Login / Sign Up


## Setup
First install all dependencies by running command:
```bash
npm install
# or
yarn install
```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
```
